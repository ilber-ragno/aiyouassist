Teste do Widget Webchat

Esta página simula um site do cliente com o widget de chat embeddable.

O balão de chat deve aparecer no canto inferior direito.

Clique no balão para abrir a janela de conversa.

Funcionalidades esperadas:

  • Balão flutuante no canto inferior direito
  • Abrir janela de chat ao clicar
  • Informar nome do visitante
  • Enviar mensagem de texto
  • Receber resposta da IA
  • Histórico mantido ao recarregar
